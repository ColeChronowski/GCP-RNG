The easiest way to run this program on app engine is to follow the AppEngine Quickstart Hello World tutorial, which can be found here: https://console.cloud.google.com/appengine/start?walkthrough_tutorial_id=python_gae_quickstart

After that tutorial is complete, replace the main.py from the tutorial with the main.py from this directory, and then run these two commands:

gcloud app create

gcloud app deploy app.yaml
